How can I capture my current process?

If you're not sure where to start in getting me a description of your current process, Windows offers an easy way to capture, export, and email a report of the steps that make up what you do. Doing this provides a quick overview that still goes into helpful details, and makes determining automation feasibility much, much easier. Additionally, it provides a great view for you into how much time each step of the way is taking. See below for instructions on how to do this (credit to @TechTipsContent)

Save and export results

After generating a report of your process, click save and give the file a name. The entire report will be saved into a ZIP file and can be emailed to me to make understanding your process and determining feasibility quicker and easier.

NOTE:
By default, a screenshot is taken each step of the way. This can be disabled by clicking the help button, settings, and selecting the "No" option. Please take care not to include any sensitive data if sharing this report.
